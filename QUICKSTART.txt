===============================================================================
  TeknoParrot Manager - HyperSpin 2 Plugin  |  v0.14.0  --  QUICK START
  Author: Jumpstile
===============================================================================

  Sets up TeknoParrot games inside HyperSpin 2 / HyperHQ, copies controls
  across games of the same type, and deploys lightgun crosshairs.
  Full details: README.txt.


  REQUIREMENTS
  ------------
  - HyperSpin 2 / HyperHQ installed and running.
  - TeknoParrot installed, with TeknoParrotUi.exe run at least once (so it
    has downloaded its GameProfiles folder).


  BEFORE YOU START
  ----------------
  - Run TeknoParrotUi.exe once so it downloads its game profiles.
  - Have at least one TeknoParrot game already extracted/installed.


  TYPICAL FIRST RUN
  ------------------
  1. Install the plugin into HyperHQ (see INSTALL THE PLUGIN below).
  2. Run the Setup Wizard.
  3. Run Health Check to see how your library looks.
  4. Preview Registration, then Register Games.
  5. In TeknoParrotUI, bind controls for one game of each type you play.
  6. Preview Control Setup, then Set Up Controls.
  7. Preview Sync, then Sync Games to add your games to HyperHQ.
  8. Launch one game from HyperHQ and test it.


  INSTALL THE PLUGIN
  -------------------
  1. Download the plugin ZIP from the Releases page.
  2. Open your HyperHQ Plugins folder.
  3. Extract the ZIP into its own folder inside Plugins -- it should end
     up as:

       Plugins\TeknoParrotManagerHyperSpin2Plugin\(files here)

     Do not extract it loose into the Plugins folder itself.

  4. Restart HyperHQ (or use "reload plugins" if your build has it).
  5. The plugin now appears in your HyperHQ plugin list.


  RUN THE SETUP WIZARD
  ---------------------
  1. Open the plugin's page inside HyperHQ.
  2. Click "Run Setup Wizard".
  3. Click Browse and point it at your TeknoParrot folder (the one
     containing TeknoParrotUi.exe) -- the wizard checks it and fills in
     the rest of the paths automatically.
  4. Follow the remaining steps. Each one explains what it's asking for in
     plain language.


  CHECK YOUR SETUP
  -----------------
  Click "Health Check" any time to see how many games are working,
  missing a profile, or have a broken path. Read-only -- changes nothing.


  REGISTER YOUR GAMES
  --------------------
  1. Click "Preview Registration" to see what can be safely registered
     without changing anything yet.
  2. Review the list, then click "Register Games" to create the missing
     profiles for real.
  3. A game not listed usually means the plugin couldn't tell which game
     it is with confidence -- set that one up by hand in TeknoParrot.


  COLLECTION DAT FILE (OPTIONAL, BUT HIGHLY RECOMMENDED)
  ----------------------------------------------------------
  Some games share one program file across dozens of unrelated titles
  (common on a few platforms), and the plugin can't always tell them
  apart on its own. The "Collection Dat" file resolves a lot of these
  automatically during registration -- most people will get noticeably
  better results with it.

  1. Click "Check Collection Dat For Updates" to see if a newer one is
     available, without downloading anything.
  2. Click "Download Collection Dat" to download it (roughly 100-150 MB)
     into the plugin's own folder.
  3. Copy the path it gives you into the "Collection Dat File" setting.
  4. Re-run "Preview Registration" -- previously unmatched games may now
     resolve automatically.

  This file is only ever read, never run or installed. It is community-
  maintained by Eggman: https://github.com/Eggmansworld/TeknoParrot


  FIX BROKEN GAME PATHS (IF NEEDED)
  -----------------------------------
  1. Click "Preview Repair" to see which broken paths can be safely fixed.
  2. Click "Repair Game Paths" to apply it.
  3. Ambiguous cases (more than one possible match) are left alone -- fix
     those by hand in TeknoParrot.


  SET UP CONTROLS (OPTIONAL BUT RECOMMENDED)
  --------------------------------------------
  1. In TeknoParrot, fully bind ONE game of each control type you use.
     Bind everything: buttons, axes, Test, Service, Coin, Start.

     Good games to bind first:
       Fighting / buttons   Street Fighter III, Tekken 7
       Driving              Daytona Championship USA, Initial D
       Lightgun             House of the Dead 4, Aliens Extermination
       Trackball            Golden Tee Live, Silver Strike Bowling

  2. Click "Device Survey" for a recommendation on which device to use
     for each game type.

  3. Click "Preview Control Setup" to see which games will receive copied
     controls and from which reference game.

  4. Click "Set Up Controls" to apply it.

  5. Launch ONE updated game and test it before trusting the rest.

  Your reference games are never changed by this step -- only the other
  games that were still unbound.


  SET UP CROSSHAIRS (LIGHTGUN GAMES ONLY, OPTIONAL)
  ----------------------------------------------------
  1. Click "Preview Crosshairs" to browse the 321 included designs in
     your browser.
  2. Pick a Player 1 and Player 2 design.
  3. Click "Deploy Crosshairs" to install them to all lightgun games.
  4. Optionally click "Hide Cursor" to hide the mouse pointer during
     lightgun play.

  Run this again any time to change your crosshair design.


  FIX GRAPHICS CARD ISSUES (OPTIONAL)
  --------------------------------------
  Some games need a setting matched to your graphics card brand to run
  correctly. The plugin can detect your card and apply this automatically.

  1. Click "Preview GPU Fix" to see which games would be changed.
  2. Click "Apply GPU Fix" to apply it. Everything is backed up first.

  Safe to run again any time you change or update your graphics card.
  Doesn't use the internet -- everything happens on your computer.


  MAKE YOUR GAMES LOOK BETTER WITH RESHADE (OPTIONAL)
  --------------------------------------------------------
  ReShade is a free tool for sharper image, CRT-style scanlines, richer
  colors, and more. You get the file yourself; the plugin installs it.

  1. Go to reshade.me and download the installer.
  2. Run it, pointing it at any 64-bit game's .exe -- it creates a DLL
     file in that game's folder.
  3. Set that DLL as the "ReShade DLL, 64-bit" setting in this plugin.
  4. Click "Check ReShade For Updates" to confirm the file is genuine.
  5. Click "Preview ReShade Setup" to see which games would be changed.
  6. Click "Apply ReShade Setup" to install it.

  Once installed, press the Home key in-game to turn effects on or off.
  To remove it from a game, delete the one file it added -- nothing else
  is touched.


  FIX OLDER GAMES THAT CRASH OR SHOW A BLACK SCREEN (OPTIONAL)
  ------------------------------------------------------------------
  Some older games crash or show a black screen on modern PCs. dgVoodoo2
  is a free tool that fixes this. You get the files yourself; the plugin
  installs them.

  1. Go to dege.freeweb.hu/dgVoodoo2/dgVoodoo2 and download it.
  2. Put the files in a folder, and set that as the "dgVoodoo2 Folder"
     setting in this plugin.
  3. Click "Preview dgVoodoo2 Setup" to see which games would be fixed --
     the plugin figures this out automatically.
  4. Click "Apply dgVoodoo2 Setup" to install it.

  Only games that actually need this are touched. To remove it, delete
  the files it added to a game's folder.


  KEEP BEPINEX UP TO DATE (OPTIONAL)
  -------------------------------------
  BepInEx is a popular modding framework some arcade game ports rely on.
  If a game already has it installed, this plugin can update it for you
  -- but it will never install BepInEx for a game that doesn't already
  have it.

  1. Click "Check BepInEx For Updates" to see which of your games with
     BepInEx already installed have a newer version available.
  2. Click "Update BepInEx" to install it. Each game's existing BepInEx
     files are backed up first.

  Downloads come straight from BepInEx's own official GitHub page --
  never a third-party mirror.


  SET UP FORCE FEEDBACK (OPTIONAL)
  -------------------------------------
  There are two separate ways to get vibration/rumble working, covering
  different games:

  FFB Blaster (TeknoParrot's own, needs a paid TeknoParrot membership to
  actually do anything):
  1. Click "Preview FFB Blaster Setup" to see which games would be
     affected.
  2. Click "Apply FFB Blaster Setup" to turn it on. Nothing is downloaded
     for this -- it's a local setting only.

  FFB Plugin (free and open-source, covers different games than FFB
  Blaster, downloaded from its own official GitHub page):
  1. Click "Preview FFB Plugin Setup" to see which games would be
     affected.
  2. Click "Apply FFB Plugin Setup" to install it.

  If a game is covered by both, FFB Blaster is used by default.


  SET UP POSTGRESQL (OPTIONAL -- ONLY NEEDED FOR A FEW SPECIFIC GAMES)
  -------------------------------------------------------------------------
  A handful of older games (Golden Tee Live, Power Putt Live, and
  similar) need a small database program called PostgreSQL to work.

  1. Click "Check PostgreSQL Status" -- if it says already installed,
     skip to step 3.
  2. Click "Install PostgreSQL" and provide the two passwords it asks
     for. Windows will ask for one-time permission to do this (a single
     permission prompt) -- nothing else in this plugin needs it. Your
     database password is saved so you won't be asked again.
  3. Click "Preview PostgreSQL Game Setup" to see which games need
     configuring, then "Apply PostgreSQL Game Setup" to do it. Everything
     is backed up first.

  Use "Backup PostgreSQL Databases" / "Restore PostgreSQL Backup" the
  same way you would for your regular profile backups. Windows only.


  ADD YOUR GAMES TO HYPERHQ
  ---------------------------
  1. Click "Preview Sync" to see what will be added to your library.
  2. Click "Sync Games" to add the "Arcade (TeknoParrot)" section with all
     your registered games.


  BACKING UP AND RESTORING
  --------------------------
  - Click "Backup Profiles" any time to save your current game profiles.
  - Click "Restore Backup" to bring back an earlier one. Your current
    profiles are backed up automatically first.


  QUICK FIXES
  ------------
  Game won't launch          Open TeknoParrotUI and point the profile to
                             the correct .exe, then re-run Repair Game
                             Paths in the plugin.

  Game not registered         Run Health Check, then Preview Registration
                             -- it usually means the plugin couldn't match
                             it with confidence. Register it by hand in
                             TeknoParrotUI.

  Controls wrong after        Restore from backup, or fix the bindings on
  control setup                your reference game in TeknoParrotUI and
                             re-run control setup.

  Game not showing up in       Run Preview Sync, then Sync Games. Make
  HyperHQ                     sure the game was registered first.

  Wrong reference game        Use the controlOverridesPath setting's
  matched                     forceArchetype entry to pin the game to the
                             correct reference game.

  Full troubleshooting: README.txt.
